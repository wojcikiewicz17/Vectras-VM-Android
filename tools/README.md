# tools/

## Objetivo
Scripts de automação para verificação estrutural, orquestração de release e inspeção de capacidades.

## Estrutura de arquivos
- `verify_repo_file_dependencies.py`: verificação de dependências de arquivos/módulos no repositório.
- `apk/`: script de orquestração de release para APK.
- `baremetal/`: scripts de integridade de diretório e detecção de capacidades de hardware.

## Conceitos principais
1. **Automação de governança**: valida coerência estrutural antes de entrega.
2. **Operação assistida**: scripts prontos para tarefas repetitivas de engenharia.
