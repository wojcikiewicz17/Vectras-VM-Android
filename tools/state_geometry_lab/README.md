# State Geometry Lab

Atualização com geometrias laterais, matrizes coexistentes, razões de Poincaré e grafo de primos na Fibonacci.

## Destaques
- Matrizes coexistentes mod[n] = `12 14 18 13 7 35 50`.
- Campo de razão com `7/3` separado de `77/33`.
- Grafo de índices primos em Fibonacci.
- Métricas laterais (lado escalado, perímetro, área, diagonal).

## Execução
```bash
python3 tools/state_geometry_lab/py/state_geometry_lab.py --demo
python3 tools/state_geometry_lab/py/state_geometry_lab.py --max 144000 --mods 64 20 18 13 12 10 14 5 4 3 2 1 0
```
