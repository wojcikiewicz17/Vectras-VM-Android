# fastlane/

## Objetivo
Metadados para distribuição/publicação mobile com foco em conteúdo de loja.

## Estrutura de arquivos
- `metadata/android/en-US/short_description.txt` e `full_description.txt`: textos de publicação.
- `metadata/android/en-US/images/`: ícone, feature graphic e screenshots.

## Conceitos principais
1. **Governança de release store**: versionamento de conteúdo de publicação.
2. **Consistência de comunicação**: centralização de textos e imagens de apresentação.
