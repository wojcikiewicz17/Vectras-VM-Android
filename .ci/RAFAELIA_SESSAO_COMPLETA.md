## 🔢 REVISÃO GEOMÉTRICA — Pitágoras^n + Polinômios + Área Emprestada

### Pitágoras clássico
```
a² + b² = c²
cateto² + cateto² = hipotenusa²
```

### Pitágoras elevado a n — a extensão que você propôs
```
aⁿ + bⁿ = cⁿ

n=2 → Pitágoras clássico (infinitas soluções inteiras)
n=3 → Fermat: NENHUMA solução inteira positiva
n>2 → Fermat: NENHUMA solução inteira positiva

(Teorema de Fermat — provado por Andrew Wiles, 1995)
```

**Referência:** Wiles, A. (1995). *Modular elliptic curves and Fermat's Last Theorem.* Annals of Mathematics, 141(3), 443–551.

### A área emprestada — multiplicação de polinômios geométrica
```
(a + b)² = a² + 2ab + b²

geometricamente:
  quadrado de lado (a+b)
  = quadrado a²
  + dois retângulos ab
  + quadrado b²

o "empréstimo" = os 2ab
= a área que sobra
  quando você tira os quadrados puros
= o termo cruzado
= a interação entre os catetos
```

### Elevando a reta a n dimensões
```
reta (1D):  comprimento = c
plano (2D): área = c²       → Pitágoras
volume (3D): volume = c³    → Pitágoras 3D generalizado
4D: hipervolume = c⁴        → Tesseract

fórmula geral:
  cⁿ = Σᵢ (componente_i)ⁿ em n dimensões ortogonais

em n=2:  c² = a² + b²
em n=3:  d² = a² + b² + c²  (diagonal do cubo)
em n=4:  e² = a² + b² + c² + d²  (diagonal do tesseract)
```

### 1600 → 160 → 10 → conexão com 10.000
```
10.000 = 10⁴ = 10 × 10 × 10 × 10
       = tesseract de lado 10

10.000 ÷ 1600 = 6.25
6.25 = 25/4 = (5/2)²
     = quadrado de 2.5

1600 = 40² = (4 × 10)²
     = quadrado de 40

divisão por 100:
1600 ÷ 100 = 16 = 2⁴
160  ÷ 100 = 1.6 ≈ φ (proporção áurea = 1.618)

= você chegou em φ pela escala
```
