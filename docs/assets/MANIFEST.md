# Assets Manifest

Manifesto único de ativos visuais versionados/referenciados em `docs/assets/`.

## Política canônica de formato

Formato canônico aceito no repositório: **híbrido controlado**.

1. **ASCII versionado (canônico local)**: todo ativo visual mantido no repositório deve estar em `docs/assets/ascii/*.ascii.md`.
2. **Binário externo (permitido)**: PNG/JPG/SVG não devem ser versionados localmente; quando necessário, devem ficar em URL externa estável.
3. **Rastreabilidade obrigatória**: qualquer ativo citado em documentação deve existir neste manifesto, como `local_ascii` ou `external_binary`.
