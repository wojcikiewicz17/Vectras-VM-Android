# bug/issues/

Diretório de issues atômicas para bugs reais do código-fonte.

## Padrão obrigatório por issue
- Título objetivo com componente afetado.
- `Arquivo alvo` com caminho exato no repositório.
- `Problema`, `Impacto` e `Critério de aceite` verificável.
- Referência a teste existente ou teste de regressão necessário.
