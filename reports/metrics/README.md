# Metrics Catalog

This directory stores the ISO 8000/9001 metrics used to guide dependency integration and quality assurance.

## Files
- `rafaelia_metrics_250.json`: 250 metrics split across ISO 8000 (data quality) and ISO 9001 (quality management).

## Usage
1. Select the metric set relevant to the change area (data quality or process quality).
2. Record the metric ID in change documentation (analysis, corrections, or test evidence).
3. Track metric impact when integrating `qemu_rafaelia` and `androidx_RmR`.
